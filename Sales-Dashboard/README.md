# 📊 Sales Follow-Up Dashboard

## Overview
An interactive CRM Sales Follow-Up Dashboard built in Microsoft Excel to monitor lead performance, revenue, follow-up activities, salesperson productivity, and business insights using dynamic PivotTables, PivotCharts.

## Features
- Interactive KPI Cards
- Dynamic PivotTables & PivotCharts
- Sales Funnel Analysis
- Revenue Trend Analysis
- Revenue by District
- Salesperson Performance
- Leads by Status
- Political Party Distribution
- Follow-up Aging Analysis
- Top Panchayaths by Revenue
- Payment Status Analysis
- Follow-up Summary

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Conditional Formatting
- Excel Formulas
- Dashboard Design
